# Amazon CoPilot Mode Example

This document demonstrates how WAP's CoPilot Mode works in practice using an Amazon product search scenario. It shows how natural language commands like "Show me 30lb dumbbells under $100 with 4+ stars" are translated into UI interactions, with detailed examples of manifest definitions for filtering, sorting, and navigating e-commerce interfaces.
